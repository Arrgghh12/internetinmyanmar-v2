ဒီခတြမှောက လူတှဟော အခွိနရြှားသထကြ ရှားလာပါတယြ။ ပှီးတော့ ကုနဈြေးနှုနြးကှီးမှင့မြှုတှအလေိုကြ စရိတစြကကလညြး ခွှတောရသေးတယလြေ။ ဒါကှောင့ြ အလုပအြကိုငြ ရှာကှတော့မယဆြိုရငြ ဟိုအရငတြုနြးကလို အိမမြှာ ဒါမှမဟုတြ မိတျတူဆိုငတြဈခုခုမှာ CV Form ကို စာစီစာရိုကလြုပြ၊ ပှီးရငြ အလုပကြှောငြှာမှာပါတဲ့ အလုပတြဈခုခငွြးစီကို လိုကတြငနြရတေဲ့ နညြးလမြးက အခွိနလြညြးကုနြ လူလညြး ပငပြနြးလှပါတယြ။ နောကပြိုငြး အလုပတြှကလေညြး ဒီလို ရုံးကို ကိုယတြိုငလြာတငထြာထကြ Email နဲ့ ပို့တာမွိုး၊ Viber ကနေ ပို့တာမွိုးကို လကခြံလာကှတဲ့အတှကြ အရငလြို ရိုးရိုးကှီး လမြးစဉကြို ကငွ့သြုံးနရငေတြော့ ခတြနေောကကြနွခြဲ့မှာပါပဲ။ဒီလို ပှောလို့ အလုပတြှကေို တဈရုံးတကြ တဈရုံးဆငြး လေ ွှာကတြဲ့သူတှေ မရှိမဟုတြ ရှိပါသေးတယြ။ ဒီလိုမွိုး လူပငပြနြးခံနမယေ့ြ အစား Online job search engines မှာ ခဏလေးအခွိနပြေးရုံနဲ့ အလုပတြှေ လှယလြှယကြူကူ လေ ွှာကနြိုငအြောငြ အလုပလြေ ွှာကလြု့ိ ရတဲ့နရောတှေ စုပှီး ဖောပြှပေးလိုကတြာလညြး ဖှဈပါတယြ။


## Myanmar Online Job Search Engines




## MyWorld


ပထမဆုံးတဈခုကတော့ Online Top Search Engine Website ဆိုရငြ [MyWorld](https://www.myworld.com.mm/) ပါပဲ၊ Myanmar’s Best Recruitment Agency လို့ ကှောငြှာထားပါတယြ။ Website ကနေ အလုပလြေ ွှာကလြို့ ရမှာ ဖှဈပှီး MyWorld Careers ဆိုတဲ့ Facebook Page နဲ့ [MyWorld Careers Co., Ltd](https://www.myworld.com.mm/) ဆိုပှီး Linkedin မှာလညြး အလုပခြေါစြာတှနေဲ့ Career Tips လို Content တှေ တငပြေးထားပါတယြ။ Link တှအေားလုံးကို စာပေါနြှိပပြှီး သှားကှည့လြို့ ရပါတယြ။

Website အနနေဲ့ဆိုရငြ MyWorld ရဲ့ Website ဟာ အမှငရြှငြးပှီး ရှာရလှယပြါတယြ။ ကိုယရြှာမယ့ြ လုပငြနြးနယပြယြ ဒါမှမဟုတြ ရာထူးနဲ့ ဆိုငတြဲ့ Keyword ကို Search Box မှာ ရှာရုံပါပဲ။



![MyWorld Website Home Page](/images/find-job-myanmar/Untitled-design-1.png)



အားသာခကွလြို့ပဲ ပှောရမယ့ြ MyWorld ရဲ့ ထူးခှားခကွကြတော့ လစာမှင့တြဲ့ အလုပအြကိုငတြှပေဲ ရှာတှေ့နိုငတြဲ့ အခကွပြါပဲ၊ အောကထြဈ လစာ ၈ သိနြးလောကကြ စတဲ့ အလုပတြှကေို ရှာတှေ့နိုငပြါတယြ။ ပှောခငွတြာက ခငဗြွားက တကယလြို့ အခုမှ ကွောငြးပှီးပှီး လုပငြနြးခှငဝြငမြယ့ြ Junior Level ဆိုရငြ MyWorld မှာ အလုပရြှာတာက ကိုယ့အြတှေ့အကှုံနဲ့ ဟနပှေီး အဆငမြပှနေိုငဘြူး ဆိုတာပါပဲ။ MyWorld နဲ့ တကယကြိုကတြာက Executive ရာထူးတှေ၊ အဲ့ဒီနောကပြိုငြး Manager နဲ့ Head of Department လို ရာထူးမွိုးတှေ ရှာဖှနတေေဲ့သူတှေ ဖှဈပါတယြ။



![MyWorldCareer LinkedIn](/images/find-job-myanmar/Untitled-design-2.png)



အလုပလြေ ွှာကပြုံအဆင့ဆြင့ကြို အကဉွြးခွုံး ပှောရရငတြော့ Account မရှိသေးရငြ Register လုပပြါ။ ပှီးရငြ သူတို့ ဖှည့ခြိုငြးတဲ့ အခကွအြလကကြို ပှည့ပြှည့စြုံစုံဖှည့ပြေးပါ။ အခုနောကပြိုငြး အလုပရြှာဖှရေေး Website တှဟော သူတို့ဆီမှာ အခကွအြလကတြှေ ဖှည့ရြုံနဲ့ အတောအြသင့ပြှည့စြုံတဲ့ CV တဈစောငြ ဖနတြီးပေးပါတယြ။ အဲဒီ CV နဲ့ ကိုယလြိုခငွတြဲ့ အလုပအြကိုငကြို ရှာပှီး Apply လုပြ လေ ွှာကရြုံပါပဲ။



![MyWorld Careers Facebook Page](/images/find-job-myanmar/Untitled-design-3.png)




## JobNet.com.mm


User တဈယောကအြနနေဲ့ နောကထြပြ ညှှနြးပေးခငွတြဲ့ အလုပအြကိုငရြှာဖှရေေး Website ကတော့ [Jobnet.com.mm](https://www.jobnet.com.mm/) ဖှဈပါတယြ။ ဘာလို့ ညှနြးပေးခငွလြဲဆိုရငြ အဲဒီ Website မှာ အလုပအြကိုငစြုံပှီး သူ့နယပြယနြဲ့သူ Filter တှေ စုံလို့ပါပဲ။



![Jobnet.com.mm Website](/images/find-job-myanmar/Untitled-design-4.png)



Filter တှမှော Industry တှေ၊ Exp level တှေ၊ လိုခငွတြဲ့ Minimum Salary နဲ့ Repat\Expat စတာတှေ ရှေးလို့ ရတဲ့အတှကြ အလုပအြတှေ့အကှုံမရှိသေးသူတှရေော၊ အလုပအြတှေ့အကှုံရှိပှီး လစာကောငြးကောငြးလိုခငွတြဲ့သူတှေ အတှကရြော အဆငပြှပေါတယြ။



![Jobnet Myanmar Facebook Page](/images/find-job-myanmar/Untitled-design-5.png)



JobNet လညြး MyWorld လိုပါပဲ Facebook Page ရှိသလို LinkedIn မှာလညြး ရှိနပေါတယြ။ Register လုပပြုံကလညြး ရိုးရှငြးပှီး CV ပါ တဈပါတညြးဆောကလြို့ ရမှာ ဖှဈပါတယြ။



![Jobnet Linkedin](/images/find-job-myanmar/Untitled-design-6.png)




## Myjobs.com.mm


“The best place to find a job” လို့ ဆိုထားတဲ့ [myjobs.com.mm](https://myjobs.com.mm/) ကတော့ Google Search Engine မှာ Myanmar Jobs ၊ Myanmar Job Search Engines စတဲ့ စကားလုံးတှနေဲ့ ရှာရငြ ထိပဆြုံးက ကလွာတတလြို့ လူသုံးလညြး မွားပါတယြ။ Website ရဲ့ Home Page မှာပဲ သကဆြိုငရြာ အလုပနြယပြယတြဈခုစီက ခေါတြဲ့ အလုပတြှကေို စာရငြး လုပပြေးထားသလို ထိပတြနြးကုမျဗဏီကှီးတှကေ အလုပအြကိုငတြှကေိုလညြး Home Page မှာပဲ ဖောပြှထားပါတယြ။ Search Bar ကနေ လိုခငွတြဲ့ အလုပအြကိုငတြှကေို ရှာရငြ လှယကြူတာကိုလညြး တှေ့ရပါတယြ။



![myjobs.com.mm website](/images/find-job-myanmar/Untitled-design-8.png)



Facebook ကနပေဲ အလုပရြှာတတတြဲ့သူတှအတှေကြ Facebook Page ရှိသလို



![myjobs.com.mm Facebook](/images/find-job-myanmar/Untitled-design-7.png)



LinkedIn မှာလညြး MyJobs Myanmar ဆိုပှီး ရှိတဲ့အတှကြ LinkedIn သုံးရငြး အလုပပြါလေ ွှာကလြို့ ရမှာပါ။ Register လုပပြုံလှယကြူပှီး CV ရှိပှီးသား ဖှဈရငလြညြး Attach တှဲပှီး တငလြို့ ရပါတယြ။



![Myjobs Myanmar LinkedIn](/images/find-job-myanmar/Untitled-design-9.png)




## MyanmarJobsDB


MyanmarJobsDB ရဲ့ ကောငြးကှကကြတော့ အလုပရြှာနတေဲ့သူတှအတှေကတြငမြဟုတဘြဲ အလုပရြှငတြှအတှေကလြညြး လိုအပတြဲ့ နရောတှအတှေကြ လူခေါဖြို့ နရောကောငြးတဈခု ဖှဈနတေဲ့အခကွပြါပဲ။ [www.myanmarjobsdb.com](https://web.archive.org/web/20210221020513/http://www.myanmarjobsdb.com/) Website မှာ Job Categories တှေ ခှဲပှီး ခေါထြားသလို စိတဝြငစြားစရာ ကောငြးတဲ့ အခကွတြှကေ Location နဲ့ ရှာနိုငတြာနဲ့ Contract ၊ Fulltime၊ Freelance၊ Part-Time နဲ့ Temporary ဆိုပှီး Filter လေးပါတဲ့အတှကြ Part-Time နဲ့ Freelance အလုပတြှကေိုပါ ရှာတှေ့နိုငမြှာပါ။ အဲ့ဒါအပှငြ ညာဘကမြှာလညြး Job Salary၊ Job Type ၊ Job Category နဲ့ Date Posted စတဲ့ Filter တှလညြေး ပါသေးတဲ့အတှကြ လစာအနညြးအမွားနဲ့လညြး ရှာလို့ ရသလို Date Posted ပါတဲ့အတှကြ လောလောလတလြတြ တငထြားတဲ့ အလုပတြှကေိုလညြး ရှာတှေ့နိုငပြါတယြ။ Facebook Page နဲ့ LinkedIn တှေ ရှိပမယေ့လြညြး Update မဖှဈတာကှာနတေဲ့အတှကြ Websites ပဲ အားကိုးရမှာပါ။

## Jobsinyangon.com


Jobsinyangon.com ကိုတော့ Google မှာ ရှာနိုငသြလို [jobsinyangon.com/app](https://web.archive.org/web/20240302003523/https://www.jobsinyangon.com/app/) ဆိုပှီးလညြး ရှာနိုငပြါတယြ။ Website ရဲ့ Home Page မှာ Job Category နဲ့ Job Location စတာတှနေဲ့ ရှာလို့ ရသလို Create your CV ဆိုတာလညြး ထငထြငရြှားရှားပါတဲ့အတှကြ Register လုပရြငြး CV ပါ တညဆြောကလြို့ ရပါတယြ။ စိတဝြငစြားဖို့ ကောငြးတဲ့အခကွကြ ဒီနေ့ အလုပဘြယနြှဈခု ရှိတယြ New Jobs Today နဲ့ Company Hiring Today စတဲ့ အခကွလြေးတှေ ပှပေးထားတာ ဖှဈပါတယြ။



![jobs in Yangon Website](/images/find-job-myanmar/Untitled-design-11.png)



Facebook မှာလညြး Jobsinyangon.com ဆိုတဲ့ Facebook Page ရှိပှီး အလုပအြကိုငတြှေ တငပြေးနပေါတယြ။ LinkedIn မှာလညြး Jobsinyangon ဆိုတဲ့နာမညနြဲ့ရှိပှီး Active ဖှဈတဲ့အတှကြ အလုပတြှေ တငထြားတာ တှေ့ရမှာ ဖှဈပါတယြ။



![Jobs in Yangon Facebook online job search engines](/images/find-job-myanmar/Untitled-design-12.png)



ဒီလောကဆြိုရငြ အှနလြိုငြးကနပှေီး မှနမြာနိုငငြံရဲ့ ထိပတြနြး Website တှမှော အလုပရြှာဖို့ တဈစိတတြဈပိုငြးအနနေဲ့ အထောကအြကူပှုလောကပြှီ ထငပြါတယြ။ စာဖတပြရိသတတြှထေဲမှာရော အလုပရြှာရငြ ဘယြ Website ဒါမှမဟုတြ ဘယြ Facebook Page တှကနေေ အလုပရြှာဖှဈသလဲ၊ ဘယြ Page တှကေို ပထမဦးစားပေးအနနေဲ့ ကှည့သြလဲဆိုတာလေး ပှောပှခဲ့ပါဦး။
