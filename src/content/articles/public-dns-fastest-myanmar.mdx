DNS, or domain name system, is the internet protocol that converts human-readable website names (such as internetinmyanmar.com) into machine readable numeric addresses also called IP addresses.




Whenever you type a URL in your web browser, whenever you open your favorite mobile app, it triggers **dozens of DNS queries** that need to be resolved before you start receiving data from the webserver.




Switching from your ISP’s offering to a custom DNS server can be highly beneficial to your Internet speed and security.




![DNS](/images/public-dns-fastest-myanmar/dns-lookup.png)




If you ever wonder **which DNS server is the fastest in Myanmar**, there is a good tool to run a quick benchmark. It is called [DNS Jumper](https://web.archive.org/web/20220913092718/https://www.sordum.org/7952/dns-jumper-v2-2/). DNS Jumper runs on Windows, it is lightweight and portable.




If you are using MacOS, you can use [Namebench](https://namebench.en.softonic.com/mac), this tool does basically the same thing. The results may vary depending on your internet service provider so try it out and adjust your DNS servers accordingly.




In Myanmar, [CloudFlare DNS](https://www.cloudflare.com/dns/) (1.1.1.1, 1.0.0.1) is one of the best choices. CloudFlare DNS servers are also good for your privacy, the company claims not to resell your data to advertisers which is not the case of all the public DNS servers.
