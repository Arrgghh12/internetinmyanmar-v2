## **You cannot use 3G or 4G Internet on your smartphone? **




### **This is a step by step guide to configure your phone to enjoy high speed 4G Internet with a Telenor Myanmar SIM Card.**


A mobile device initiating a data connection must be configured with an APN or Access Point Name to present to the mobile operator. The carrier will then review this APN identifier to decide what type of network connection should be created, for example: which IP addresses should be assigned to the customer device, which security methods should be used, and how or if, it should be connected to some private or public network.





If you are interested to see Telenor 2G, 3G, 4G and 5G mobile network coverage in Myanmar, take a look at [nPerf Online Map](https://www.nperf.com/en/map/MM/-/17442.ATOM-Mobile/signal?ll=18.263215257495162&lg=103.89999999999999&zoom=6).


### **Telenor APN Configuration on Android: **





Tap Settings | More | Wireless & Networks | Mobile Networks, and then tap Access Point Names
From this window, tap the plus sign [+]. You should now be able to add the information for the new APN


### **Telenor APN Configuration on Xiaomi MIUI:**


Tap Settings
Tap SIM cards & mobile networks
Tap Ooredoo
Tap Access Point Names
Tap New APN


### **Telenor APN settings:**


Name - Telenor Internet
APN   - internet
Proxy - not set
Port    - not set
Username - not set
Password - not set
Server - not set
MMSC - not set
MMS Proxy - not set
MMS Port - not set
MCC - 414
MNC - 05
Authentication type - CHAP or PAP
APN type - Default
APN protocol - IPv4
APN roaming protocol - IPv4
APN enable/dissable - APN enabled
Bearer - Unspecified
Promtp password - not set
