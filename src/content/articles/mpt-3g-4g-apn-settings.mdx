## **Not able to use 3G or 4G Internet on your smartphone in Myanmar?**




**Find below our step-by-step guide to configure your phone to enjoy high speed 4G Internet with an MPT SIM Card.**




If you have just bought a SIM, the first thing you need to do is to configure MPT APN settings. This guide below will help you enjoy MPT 4G Internet within a few minutes!




If you are interested to see MPT 2G, 3G, 4G and 5G mobile network coverage in Myanmar, take a look at [nPerf Online Map](https://www.nperf.com/en/map/MM/-/4980.MPT-Mobile/signal?ll=18.263215257495162&lg=103.89999999999999&zoom=6).




### What is an APN?




A mobile device initiating a data connection must be configured with an APN or Access Point Name to present to the mobile operator. The carrier will then review this APN identifier to decide what type of network connection should be created, for example: which IP addresses should be assigned to the customer device, which security methods should be used, and how or if, it should be connected to some private or public network.




This is our step-by-step guide to enjoy high speed 4G Internet with a MPT SIM:




### MPT Internet Setting for Android:




#### Smartphones with Android v 5.0 or earlier:




APN settings can be set in your Apps under:




- Settings
- More Networks (or Cellular Networks)
- Mobile Networks (or Wireless and Networks)
- Access Point Names
- Add New APN




**Name:**  MPT
**APN:** mptnet
*Leave all other fields empty including the Username and Password as these are not required.




#### Smartphones with Android v 6.0 or above:




- From the home screen, tap the Menu button.
- Tap **Settings**
- Tap **Mobile networks**
- Tap **Access Point Names**
- Tap the **Menu** button
- Tap **New APN**
- Tap the **Name** field
- Enter MPT, then tap **OK**
- Tap the **APN** field
- Enter **mptnet**, then tap **OK**
- Tap the **Menu** icon
- Tap **Save**




### MPT Internet Setting for Xiaomi MIUI:




- Tap Settings
- Tap SIM cards & mobile networks
- Tap MPT
- Tap Access Point Names
- Tap New APN
- Tap the **Name** field
- Enter **MPT**, then tap **OK**
- Tap the **APN** field
- Enter **mptnet**, then tap **OK**
- Type **More**
- Type **Save**

### MPT Internet Setting for iOS (iPhone & iPad):




- Select Settings
- Select Mobile Data
- Select Mobile Data Options
- Select Mobile Data Network
- Scroll to and select Reset Settings
- Under the **Mobile /** **Cellular Data **section change the **APN **to the word **mptnet **(leaving Username and Password blank)
- Your Iphone is now easy to use MPT 4G
