ဖုန်းပျောက်ပြီဆိုရင် ကျွန်တော်တို့တွေ ဘာလုပ်လို့ဘာကိုင်ရမှန်းမသိဖြစ်တတ်ပါတယ်။ အဲ့ဒါကြောင့် ဒီတစ်ခေါက်မှာတော့ ကျွန်တော်တို့ Internet In Myanmar က android ဖုန်းသုံးသူတွေအတွက် ဖုန်းပျောက်တာနဲ့ ဘာတွေလုပ်ရမလဲ ။ ဘယ်လိုတွေရှာရမလဲဆိုတာကို ပြောပြပေးသွားပါမယ်။


**Find My Device Application **




ဒီ application ကတော့ ကျွန်တော်တို့မဖြစ်မနေသိထားရမယ့် သုံးရမယ့် application တစ်ခုပဲဖြစ်ပါတယ်။




![Myanmar internet censorship](/images/find-android-myanmar/Find-My-Device-App.png)


**Find My Device ကို ဖုန်းမှာသုံးလို့ရမရ ဘယ်လိုသိနိုင်မလဲ။**


အရင်ဆုံး အဲ့ဒီ application ကကျွန်တော်တို့ဖုန်းနှင့် အဆင်ပြေမပြေ သိဖို့လိုအပ်ပါတယ်။ အနိမ့်ဆုံး Andriod Version 4.0 ရှိဖို့လိုပါတယ်ခင်ဗျာ။ အဲ့ဒါကြောင့် ကမ္ဘာ့ပေါ်မှာရှိတဲ့ Andriodဖုန်းသုံးသူတွေရဲ့ ၉၉% (၂ ဘီလီယံလောက်) ထိသုံးလို့ရတယ်လို့ ယူဆလို့ရပါတယ်။


**Find My Device ကို အသုံးပြုပုံအဆင့်ဆင့်**

အလွယ်ဆုံးအဆင့်ကနေစကြရအောင်။

အရင်ဆုံး **Google Play Store** ကနေ **Find My Device **ဆိုတဲ့ application ကိုရှာမယ်။

Install လုပ်မယ်။ ပြီးရင်တော့ ဒီအောက်ကအဆင့်တွေကို တစ်ဆင့်ချင်းလုပ်ဖို့လိုပါတယ် ခင်ဗျာ။

ဘာတွေလဲဆိုတော့…




၁။ Application Install လုပ်ပြီးသွားရင်တော့ နောက်တစ်ဆင့် အနေနဲ့ Find My Device Account ဖွင့်ရပါမယ်။ ထုံးစံအတိုင်းအကောင့်ဖွင့်ဖို့ အတွက် Email account တစ်ခုလိုပါတယ်ခင်ဗျာ။




၂။ တကယ်လို့အကောင့်ရှိပြီးသားဆိုရင်တော့ Email Address ရေးပြီး Password ထည့်လိုက်ရင်ရပါပြီ။ Sign In As Guest ဆိုတာကိုရွေးလိုက်ရင်တော့ New Email Account ဆိုပြီးပေါ်လာမှာဖြစ်ပါတယ်။ အဲ့လိုပေါ်လာရင်တော့ Email Address နဲ့ Password လိုက်ရင်ရပါပြီ။




၃။ ပြီးရင်တော့ Sign In ကိုနှိပ်လိုက်ရင် အကောင့်ဝင်ပြီးသားဖြစ်သွားမှာပါ ။




၄။ တစ်ခုရှိတာက Sign In ဝင်ပြီးသွားရင် Location Access ကို Allow လုပ်ထားဖို့လိုပါတယ်။


**ဖုန်းကို ရှာလို့ရမရကော ဘယ်လိုသိနိုင်မလဲ။**


Find My Device Application ကို Sign In ဝင်ပြီး Location Access ပေးလိုက်တာနဲ့ တပြိုင်နက် ဖုန်းရဲ့အမျိုးစားနှင့် လက်ရှိတည်နေရာကိုပြနေမှာပဲဖြစ်ပါတယ်။




ပြီးရင်တော့ ရွေးချယ်စရာအနေနဲ့ ၃ ခုရှိပါတယ်။


၁။ ဖုန်းကို ၃၅ မိနစ်တိတိ အသံမြည်အောင် လုပ်ပေးနိုင်မယ့် **Play Sound** ကိုရွေးမလား။


၂။ **Secure Device** ကိုရွေးပြီးတော့ ဖုန်းကို **Lock** ချမလား။ Lock ချမယ်လို့ စဉ်းစားထားတယ်ဆိုရင်တော့ ဖုန်းရဲ့ Lock Screen မှာစာပေါ်အောင် Lock Screen Message ချန်တာပဲဖြစ်ဖြစ် ဖုန်းနံပါတ်ချန်တာပဲဖြစ်ဖြစ် ရေးပြီးတော့ Secure Device ကိုနှိပ်လိုက်ရင်ရပါပြီ။




၃။ ဒါမှမဟုတ် **Erase Device **ကိုရွေးပြီး ဖုန်းထဲကအချက်အလက်တွေကို ဖျက်ပစ်မလား။




(ဒါပေမယ့် Erase Device ကတော့ဖုန်းပျောက်သွားတဲ့ အခါမှပဲသုံးလို့ရမှာပါ တကယ်လို့ လက်ရှိသုံးနေတဲ့ဖုန်းဆိုရင်တော့ အဲ့ Function ကိုမသုံးသင့်ပါဘူးခင်ဗျာ။)

**Find My Device ကို Website ကနေဘယ်လိုရှာကြမလဲ။**


၁။ အရင်ဆုံး [https://www.google.com/android/find?u=0](https://www.google.com/android/find?u=0) ကိုနှိပ်ပါ။



![Myanmar internet censorship](/images/find-android-myanmar/Find-My-Device-Desktop-1.png)




၂။ ပြီးရင်ခုနက application မှာလိုပဲ Email, Password ထည့်ပြီး Sign In နှိပ်လိုက်ရင် ရပါပြီ။




၃။ နောက်ဆုံးအဆင့် အနေနဲ့ ခုနက Function ၃ ခုကိုတွေ့ရမှာပဲဖြစ်ပါတယ်ခင်ဗျာ။




တစ်ခုသတိထားရမှာက တစ်ဖက်လူကဖုန်းမပိတ်ထားခင်သာ ဖုန်းကို အသံမြည်အောင်လုပ်လို့ရမှာပဲ ဖြစ်ပါတယ်ခင်ဗျာ။ ပျောက်နေတဲ့ဖုန်းကိုပြန်မဖွင့်မချင်းအသံမြည်မှာမဟုတ်ပါဘူး။ တကယ်လို့ဖုန်းပိတ်ထားတယ်ဆိုရင်တော့  Secure Device ကိုနှိပ်ပြီး ဖုန်းကို လော့ချတာပဲဖြစ်ဖြစ် ဖုန်းထဲက Data တွေကို Erase Device ကိုနှိပ်ပြီးဖျက်ပစ်လို့တော့ရပါတယ်။
