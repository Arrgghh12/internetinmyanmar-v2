### What is an APN?




A mobile device initiating a data connection must be configured with an APN or Access Point Name to present to the mobile operator. The carrier will then review this APN identifier to decide what type of network connection should be created, for example: which IP addresses should be assigned to the customer device, which security methods should be used, and how or if, it should be connected to some private or public network.




This is our step by step guide to enjoy high speed 4G Internet with a 
Ooredoo SIM:




### Ooredoo Internet Setting for Android:




#### Smartphones with Android v 5.0 or earlier:




APN settings can be set in your Apps under:




- Settings
- More Networks (or Cellular Networks)
- Mobile Networks (or Wireless and Networks)
- Access Point Names
- Add New APN




**Name:**  Ooredoo
**APN:** internet
*Leave all other fields empty including the Username and Password as these are not required.




#### Smartphones with Android v 6.0 or above:




- From the home screen, tap the Menu button.
- Tap **Settings**
- Tap **Mobile networks**
- Tap **Access Point Names**
- Tap the **Menu** button
- Tap **New APN**
- Tap the **Name** field
- Enter Ooredoo, then tap **OK**
- Tap the **APN** field
- Enter **internet**, then tap **OK**
- Tap the **Menu** icon
- Tap **Save**




### Ooredoo Internet Setting for Xiaomi MIUI:




- Tap Settings
- Tap SIM cards & mobile networks
- Tap **Ooredoo**
- Tap Access Point Names
- Tap New APN
- Tap the **Name** field
- Enter **Ooredoo**, then tap **OK**
- Tap the **APN** field
- Enter **internet**, then tap **OK**
- Type **More**
- Type **Save**

### Ooredoo Internet Setting for iOS (iPhone & iPad):




- Select Settings
- Select Mobile Data
- Select Mobile Data Options
- Select Mobile Data Network
- Scroll to and select Reset Settings
- Under the **Mobile /** **Cellular Data **section change the **APN **to the word **internet **(leaving Username and Password blank)
- Your Iphone is now easy to use Ooredoo 4G
