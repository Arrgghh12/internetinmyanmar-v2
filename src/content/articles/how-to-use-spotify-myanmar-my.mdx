ပွီးခဲ့တဲ့ရကျပိုငျးက ကြှနျတောျတို့ က VPN ဘယျလိုကြောျမလဲဆိုတာနဲ့ပတျသကျပွီးပွောပွခဲ့ပါတယျ။ ဒီတဈခေါကျမှာတော့ VPN ကြောျပွီး ကမ်ဘာပတျရငျး သီခငြျးနားထောငျကွရအောငျ။

သီခငြျးခစြျသူတှအေတှကျ အခြိနျနဲ့ တပွေးညီနားထောငျလို့ရတဲ့ ဆှီဒငျနိုငျငံထုတျ ကမ်ဘာ့ထိပျတနျး app တဈခုအကွောငျး ကိုပွောပွပေးခငြျပါတယျ ။ လကျရှိ အသုံးပွုသူပေါငျး သနျး ၁၀၀ ကြောျထိ ရှိနတေဲ့ application တဈခုဖွဈရုံသာမက သီခငြျးပုဒျရပေေါငျး သနျး ၃၀ ကြောျ အထိရှိပါတယျ။

နောကျထပျ Spotify ရဲ့အားသာခကြျကတော့ ကှနျပြုတာမှာပဲဖွဈဖွဈ ဖုနျးမှာပဲဖွဈဖွဈ သုံးလို့ရပါတယျ။

အကောင့ျဖှင့ျမယျဆိုရငျတော့  မွနျမာနိုငျငံ မှာ လကျတလော ဖှင့ျလို့မရသေးပါဘူး အဲ့ဒါကွောင့ျ ထုံးစံအတိုငျး VPN ကြောျပွီးဖှင့ျမှရမှာပါ ။

 


## Spotify ကို Windows Laptop မှ အကောင့ျဖှင့ျနညျး


www.spotify.com ကိုဝငျလိုကျပွီဆိုတာနဲ့ Spotify is not available in Myanmar ဆိုတဲ့ စာသားက စောင့ျနတေော့မှာပဲဖွဈပါတယျ။
[![Spotify not available in Myanmar message](/images/how-to-use-spotify-myanmar-my/Not-Available.png)](/wp-content/uploads/2016/11/Not-Available.png)
_Spotify is not available in Myanmar_

အဲ့တော့ဟိုတဈနေ့တုနျးက ကြှနျတောျပွောသလိုပဲ VPN ကြောျရပါမယျ ။ ဒါပမေယ့ျ VPN မကြောျခငျ spotify သုံးလို့ရတဲ့နိုငျငံတှကေို [ဒီနရော](https://www.spotify.com/is/select-your-country/)မှာ ကွည့ျလို့ရပါတယျ ခငျဗြာ။

VPN software ကို စပွီးအသုံးပွုလိုကျပွီဆိုတာနဲ့ Google ကကြှနျတောျတို့ ရှိနတေဲ့ နိုငျငံအမှနျကို မသိရတော့ပါဘူး။ ဒါ့အပွငျ VPN ကို ခလုတျလေးတဈခကြျနှိပျရုံနဲ့ အလှယျတကူ ဖှင့ျလို့ပိတျလို့ရပွီး အခြိနျတိုငျးဖှင့ျထားစရာလညျး မလိုပါဘူး။ တကယျလို့ မွနျမာနိုငျငံမှာ မရနိုငျတဲ့ application တှကေို download လုပျမယျဆိုမှသာ ဖှင့ျထားရုံပါပဲ ။

ဒီအောကျက software ၂ ခုက ကြှနျတောျသုံးလေ့ရှိတဲ့ စိတျခရြတဲ့ VPN software တှပေါ။

[TunnelBear](https://www.tunnelbear.com/) ဆိုတဲ့ Canada က VPN software application အကွောငျးကို အရငျဆုံးပွောပွခငြျပါတယျ။ ဒီ software ကိုသုံမယျဆိုရငျတော့ တနေ့ကို 500MB လောကျ VPN ကြောျပွီးသုံးလို့ရပါတယျ။ ပွောရမယျဆိုရငျ 500 MB ဆိုတာ သီခငြျးတှေ download လုပျဖို့အတှကျတော့ တောျတောျကွီးကို လုံလောကျနပေါပွီ။

နောကျတဈခုကတော့ [Windscribe](https://windscribe.com/) VPN Software ပါ။ သုံးရတာလှယျကူပွီး တဈလကို 10GB အထိ ဝနျဆောငျခပေးစရာမလိုဘဲ သုံးလို့ရပါတယျ။

တကယျလို့ Windscribe ကိုသုံးမယျဆိုရငျတော့ Windscribe software ကို [Play Store](https://play.google.com/store/apps/details?id=com.windscribe.vpn&hl=en) ကနေ download လုပျပွီး install လုပျပါ။

ပွီးရငျ အကောင့ျဖှင့ျပါ။ အဲ့ဒါပွီးရငျတော့ ပုံကအတိုငျး သင့ျတောျတဲ့နိုငျငံ ကိုရှေးလို့ရပါပွီ ။ (ဥပမာ – United States)

Spotify ကိုဝငျလိုကျပွီဆိုရငျတော့ ကိုယျကွိုကျနှဈသကျရာအဆိုတောျတှေ သီခငြျးအမြိုးစားတှကေိုရှေးခယြျရပါမယျ။ အဲ့ဒီ့အဆင့ျတှပွေီးရငျတော့ ကြှနျတောျတို့ Spotify ကနေ ကမ်ဘာကြောျပွီး  မွနျမာနိုငျငံမှာမရနိုငျသေးတဲ့ သီခငြျးတှကေို ပြောျပြြောျကွီးနားထောငျလို့ရပါပွီ။


## Smartphone ကနေ Spotify အကောင့ျဖှင့ျနညျး


ပထမဆုံး Wind Scribe application ကို VPN ကြောျဖို့ download လုပျရပါမယျ။

ပွီးရငျ အကောင့ျဖှင့ျပွီး ပုံကအတိုငျးနိုငျငံရှေးပါ။



![VPN Myanmar Google Play Store Netflix Spotify](/images/how-to-use-spotify-myanmar-my/Windscribe-VPN-Myanmar.png)



VPN Software ကိုသုံးလိုကျပွီဆိုတာနဲ့ Spotify က ကြှနျတောျတို့ ရှေးထားတဲ့နိုငျငံကို ကြှနျတောျရောကျနတေဲ့ နိုငျငံလို့ထငျသှားမှာပဲဖွဈပါတယျ။

VPN ကြောျပွီးပွီဆိုရငျတော့ Browser ကနေ spotify.com ကိုဝငျ ၊ ပွီးရငျ Spotify Application ကို download လုပျပါ။

အဲ့ဒါပွီးရငျတော့ Spotify account ဖှင့ျပါ။ ဒါပမေယ့ျ VPN ကို တော့ account ဖှင့ျပွီးတဲ့အထိ မပိတျသင့ျပါဘူးခငျဗြာ။

အခုအဆင့ျတှပွေီးရငျတော့ ကမ်ဘာကြောျသီခငြျးတှကေို ပြောျပြောျကွီးနားထောငျလို့ရပါပွီ ခငျဗြာ။

တဈခုသတိထားရမှာက Windscribe ကို သီခငြျးနားထောငျနတေဲ့အခြိနျမှာတော့ ဖှင့ျထားစရာမလိုပါဘူး။ တကယျလို့ မွနျမာနိုငျငံမှာ မရနိုငျတဲ့ application တှေ၊ သီခငြျးတှကေို download လုပျမယျဆိုမှသာ ဖှင့ျထားရုံပါပဲ ။
