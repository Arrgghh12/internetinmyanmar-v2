### Mytel, the 4th network operator in Myanmar launches 2G and 4G service in 2018. As of January 2019, Mytel has around five million subscribers.




Mytel SIM Cards are available nationwide in Myanmar. Mytel will allow you to enjoy high speed 4G Internet in Myanmar main cities.




If you are interested to see Mytel 2G, 3G, 4G and 5G mobile network coverage in Myanmar, take a look at [nPerf Online Map](https://www.nperf.com/en/map/MM/-/137098.Mytel/signal/?ll=19.34388685108229&lg=96.685&zoom=5).




If you have just bought a SIM, the first thing you need to do is to configure Mytel APN settings on your handset. This guide below will help you enjoy Mytel 4G Internet within a few minutes!




### What is an APN?




A mobile device initiating a data connection must be configured with an APN or Access Point Name to present to the mobile operator. The carrier will then review this APN identifier to decide what type of network connection should be created, for example: which IP addresses should be assigned to the customer device, which security methods should be used, and how or if, it should be connected to some private or public mobile network.




This is our step by step guide to enjoy high speed 4G Internet with a MYTEL SIM Card:




### Mytel Internet Setting for Android (Prepaid and Postpaid):




#### Smartphones with Android v 5.0 or earlier:




APN settings can be set in your Apps under:




- Settings
- More Networks (or Cellular Networks)
- Mobile Networks (or Wireless and Networks)
- Access Point Names
- Add New APN




**Name:**  Mytel
**APN:** Mytel
*Leave all other fields empty including the Username and Password as these are not required.




#### Smartphones with Android v 6.0 or above:




- From the home screen, tap the Menu button.
- Tap **Settings**
- Tap **Mobile networks**
- Tap **Access Point Names**
- Tap the **Menu** button
- Tap **New APN**
- Tap the **Name** field
- Enter **Mytel**, then tap **OK**
- Tap the **APN** field
- Enter **Mytel**, then tap **OK**
- Tap the **Menu** icon
- Tap **Save**




### Mytel Internet Setting for Xiaomi MIUI:




- Tap Settings
- Tap SIM cards & mobile networks
- Tap Mytel
- Tap Access Point Names
- Tap New APN
- Tap the **Name** field
- Enter **Mytel**, then tap **OK**
- Tap the **APN** field
- Enter **Mytel**, then tap **OK**
- Type **More**
- Type **Save**

### Mytel Internet Setting for iOS (iPhone & iPad):




- Select Settings
- Select Mobile Data
- Select Mobile Data Options
- Select Mobile Data Network
- Scroll to and select Reset Settings
- Under the **Mobile /** **Cellular Data **section change the **APN **to the word **Mytel **(leaving Username and Password blank)
- Your Iphone is now easy to use Mytel 4G
